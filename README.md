# Pittsburgh-Traffic-Flow
I commuted to college and now really want to know if the real numbers line up with what I experienced.

## Data Sources — Week 2 (Traffic Flow)

- **Road network & hierarchy**: © OpenStreetMap contributors, available under the
  [Open Database License (ODbL)](https://opendatacommons.org/licenses/odbl/). Data accessed
  programmatically via the [`osmnx`](https://github.com/gboeing/osmnx) Python library
  (Boeing, G. 2017. "OSMnx: New Methods for Acquiring, Constructing, Analyzing, and
  Visualizing Complex Street Networks." *Computers, Environment and Urban Systems* 65,
  126–139).

- **Commuting flows**: U.S. Census Bureau, Center for Economic Studies, *LEHD Origin-Destination
  Employment Statistics (LODES)*, dataset version LODES8, Pennsylvania, 2022. Accessed via
  direct file download from https://lehd.ces.census.gov/data/. Public domain (U.S. federal
  government data).

- **Tract boundaries**: U.S. Census Bureau, *TIGER/Line Shapefiles*, accessed via the
  [`pygris`](https://github.com/walkerke/pygris) Python library. Public domain.

- **Basemap tiles**: © OpenStreetMap contributors, © [CARTO](https://carto.com/attributions)
  (Positron basemap style), accessed via the [`contextily`](https://github.com/geopandas/contextily)
  Python library.
